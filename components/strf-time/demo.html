<time is="strf-time"> Demo

Note that the datetime attribute must always be set to a valid date with an optional time string. Examples of using <time is="strf-time">:

With no format or locale attribute, it defaults to %Y-%m-%d.

Specified format="%A, %B %e", but relying on browser's language setting to localize weekday and month.

Specified format="%A, %B %e" and locale="de", then locale="es" to localize weekday and month.
